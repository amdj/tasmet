TaSMET

Thermoacoustic System Modeling Environment Twente

Welcome to TaSMET, the Thermoacoustic System Modeling Environment Twente. TaSMET is a computer code that is able to simulate thermoacoustic (TA) and Stirling systems. TaSMET provides nonlinear models to simulate laminar oscillating flow in ducts, heat exchangers, stacks and pistons. These nonlinear models can be connected to form a model of a complete TA or Stirling system.
